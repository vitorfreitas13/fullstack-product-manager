🚀 Fullstack Product Manager

Projeto Fullstack para gerenciamento de produtos, permitindo cadastrar, listar, editar e deletar produtos.

Aplicação desenvolvida com React no frontend e Node.js + Express no backend, com deploy em nuvem.

📡 Endpoints da API
📥 Listar produtos
GET /produtos
➕ Criar produto
POST /produtos
✏️ Atualizar produto
PUT /produtos/:id
❌ Deletar produto
DELETE /produtos/:id
🧠 Como rodar o projeto localmente
🔹 Backend
cd back
npm install
node server.js

Servidor disponível em:

http://localhost:5000
🔹 Frontend
cd front
npm install
npm run dev
